Learning project to build Nintendo Entertainment System emulator.

## Controls

`Arrow keys` - D-pad<br>
`S` - Start<br>
`A` - Select<br>
`Z` - Button B<br>
`X` - Button A<br>
`R` - Reset<br>
`Esc` - Quit
